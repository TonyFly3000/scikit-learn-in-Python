# if/elif/else


```{python}
a = 200
b = 33
if b > a:
  print("b is greater than a")
elif a == b:
  print("a and b are equal")
else:
  print("a is greater than b")
```

# Loop

## for Loop

```{python}
fruits = ["apple", "banana", "cherry"]
for x in fruits:
  print(x)
```

## while Loop

```{python}
i = 1
while i < 4:
  print(i)
  i += 1
```

with break statement

```{python}
i = 1
while i < 6:
  print(i)
  if i == 3:
    break
  i += 1
```
