1.2.11 list to dictionary

1.2.11.1 one list to dictionary

Code 
list=['a', 1, 'b', 2, 'c', 3]

def convert(lst):
   res_dict = {}
   for i in range(0, len(lst), 2):
       res_dict[lst[i]] = lst[i + 1]
   return res_dict
 
convert(list)
{'a': 1, 'b': 2, 'c': 3}
